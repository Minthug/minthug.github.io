개인정보처리방침

치지직 채팅 분석기 (Chzzk Chat Analyzer) Chrome 확장 프로그램
최종 수정일: 2026년 2월 27일

이 확장 프로그램은 수집한 데이터를 외부 서버로 전송하지 않습니다.
모든 데이터는 사용자의 브라우저 내 로컬 저장소에만 저장됩니다.

1. 수집하는 정보

확장 프로그램은 치지직(chzzk.naver.com) 페이지에서 아래 정보를 수집합니다.

  • 채팅 메시지 텍스트 및 닉네임 (키워드 급증 감지 목적)
  • 채팅 발생 수량 및 타임스탬프 (급증 구간 분석 목적)
  • VOD/라이브 영상 재생 위치 (시간 기반 분석 목적)
  • 스트리머 채널명 및 방송 제목 (파일명 생성 목적, chzzk 공개 API 사용)

2. 정보 사용 목적

  • 채팅 급증 구간(편집 포인트) 자동 감지 및 표시
  • 키워드 빈도 급증 구간 분석
  • 매니저 채팅 별도 수집 및 표시
  • 분석 결과 TXT / CSV 파일 내보내기

3. 데이터 저장 방식

  • 모든 데이터는 사용자 브라우저의 chrome.storage.local에만 저장됩니다.
  • 외부 서버, 데이터베이스, 클라우드 서비스로 전송되지 않습니다.
  • 확장 프로그램 삭제 시 모든 저장 데이터가 함께 삭제됩니다.
  • 세션 초기화 또는 영상 변경 시 이전 세션 데이터가 자동으로 삭제됩니다.

4. 제3자 정보 제공

수집된 어떠한 정보도 제3자에게 제공, 판매, 공유되지 않습니다.

5. 외부 API 사용

스트리머명 및 방송 제목 자동 인식을 위해 치지직 공개 API (api.chzzk.naver.com)를 호출합니다. 이 요청은 사용자가 이미 로그인한 치지직 계정의 쿠키를 사용하며, 별도의 개인정보를 추가로 전송하지 않습니다.

6. 권한 사용 목적

  • tabs: 현재 활성 탭의 URL 및 타이틀 확인
  • storage: 분석 세션 데이터 로컬 저장
  • downloads: TXT / CSV 파일 내보내기
  • scripting: 치지직 페이지에 채팅 감지 스크립트 삽입
